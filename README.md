# 📱 Mobile Tetris (HTML / JavaScript)

모바일과 PC에서 모두 플레이 가능한 **HTML5 테트리스 게임**입니다.  
터치 조작을 지원하며, 스마트폰 세로 화면에 최적화된 UI를 제공합니다.

---

## 🎮 주요 기능

- 🎲 클래식 테트리스 게임 플레이
- 📱 모바일 터치 컨트롤 지원 (凸자 형태 버튼)
- 💻 키보드 조작 지원 (PC)
- 🔄 블록 회전 / 이동 / 빠른 낙하
- 🧩 라인 클리어 애니메이션
- ❌ 게임 오버 화면 및 재시작

---

## 📐 조작 방법

### 모바일 (터치)


### PC (기본 키)
- ← → : 좌우 이동
- ↓ : 아래로 이동
- ↑ : 회전
- Space : 하드 드롭

---

## 🚀 실행 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 모바일에서는 전체 화면으로 플레이하는 것을 권장합니다.
